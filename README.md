# CP949 Encoder

CP949 Encoder is a JavaScript utility library designed to encode strings using Code Page 949 (CP949). CP949 is a character encoding system used to represent Korean text in computing, extending the EUC-KR encoding with additional characters.

## Examples

```tsx
import { encodeToCp949 } from "cp949-encoder";

console.log(encodeToCp949("안녕하세요"));
// Output: BEC8B3E7C7CFBCBCBFE4
```
